# 🎼 SynchroWork (人机协奏)

智能人机作业分析引擎 - 基于工业工程理论的生产仿真工具

---


## 项目简介

SynchroWork 是一款专为工业工程师（IE）打造的人机作业分析工具。它将复杂的生产要素"编排"为流畅的作业乐章，
通过可视化仿真帮助用户识别生产过程中的隐形浪费，优化资源配置。

## 核心功能

### 1. 参数化建模
• 定义作业步骤（名称、类型、时长）
• 配置资源数量（操作员、设备）
• 设置行走时间等全局参数

### 2. 可视化仿真
• 实时甘特图展示人机作业配合
• 支持倍速播放和快进完成
• 动态追踪在制品状态

### 3. 性能分析
• 自动计算周期时间（CT）和产能（UPH）
• 分析资源利用率（人员、设备）
• 识别瓶颈资源（人是瓶颈/机是瓶颈）
• 统计等待时间（人等机/机等人）

### 4. 多场景支持
• 1人1机基础场景
• 1人N机复杂场景
• 支持联合作业、纯人工、纯设备三种作业类型

## 应用场景

### 产线规划
在新产线投产前，通过仿真确定最优的设备数量和人员配置，避免资源浪费或产能不足。

### 产能分析
对现有产线进行建模，识别瓶颈环节，评估改善方案的效果。

### 方案对比
对比不同配置方案（如增加设备、调整作业顺序），选择投资回报率最高的方案。

### 教学培训
直观展示人机协作原理，帮助学员理解工业工程中的时间研究和动作分析。

## 使用方式

### 在线访问
无需安装，浏览器直接访问即可使用。

### 基本流程
1. 选择行业案例或手动配置资源
2. 定义作业步骤序列
3. 启动仿真或快进完成
4. 查看甘特图和性能指标
5. 根据分析结果调整配置

### 交互特性
• 拖拽调整作业步骤顺序
• 滑块实时修改作业时长
• 内联编辑作业参数
• 支持清空和重置操作
